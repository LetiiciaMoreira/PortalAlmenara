# 🌐 Portal Almenara

Projeto desenvolvido com o objetivo de centralizar e disponibilizar informações relevantes sobre a cidade de Almenara, proporcionando uma experiência simples, intuitiva e acessível aos usuários.

## 📋 Sobre o projeto

O Portal Almenara foi criado para reunir conteúdos e informações importantes em uma única plataforma, facilitando o acesso dos usuários e promovendo maior organização e praticidade na navegação.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## ✨ Funcionalidades

- Interface moderna e responsiva;
- Navegação simples e intuitiva;
- Organização das informações em diferentes seções;
- Experiência adaptada para dispositivos móveis;
- Layout desenvolvido com foco na usabilidade.

## 💻 Minha participação

Este projeto foi desenvolvido aplicando conhecimentos em desenvolvimento web, com foco na construção da interface, estilização das páginas e implementação das funcionalidades utilizando JavaScript.

Durante o desenvolvimento, foram trabalhados conceitos como:

- Estruturação de páginas com HTML;
- Estilização e responsividade com CSS;
- Manipulação do DOM com JavaScript;
- Organização e manutenção do código;
- Boas práticas de desenvolvimento web.
